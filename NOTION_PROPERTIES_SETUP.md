# Notionデータベース プロパティ設定手順

## 必要なプロパティ

Notionデータベースに以下のプロパティ（列）を追加してください：

### 1. 日付
- **タイプ**: Date
- **名前**: `日付`（全角）
- **設定**: Start dateのみ

### 2. カテゴリ
- **タイプ**: Multi-select
- **名前**: `カテゴリ`（全角）
- **オプション**:
  - 実装
  - 学習
  - バグ修正
  - リファクタリング
  - ドキュメント
  - テスト
  - 設定

### 3. ステータス
- **タイプ**: Select
- **名前**: `ステータス`（全角）
- **オプション**:
  - 完了
  - 進行中
  - 実装中

### 4. 所要時間
- **タイプ**: Number
- **名前**: `所要時間`（全角）
- **単位**: 時間（h）

## 設定手順

1. Notionデータベースページを開く
2. 右上の「...」メニューから「プロパティ」を選択
3. 「+ 新しいプロパティ」をクリック
4. 上記の設定に従って4つのプロパティを追加
5. 保存して完了

## 確認方法

プロパティが正しく追加されると、データベースのテーブルビューで以下の列が表示されます：
- Name（既存）
- 日付
- カテゴリ
- ステータス
- 所要時間

## 注意事項

- プロパティ名は**全角**で正確に入力してください
- Multi-selectとSelectの違いに注意
  - Multi-select: 複数選択可能（カテゴリ用）
  - Select: 単一選択のみ（ステータス用）
